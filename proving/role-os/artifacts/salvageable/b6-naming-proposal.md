## Naming Proposal: User-Friendly Terminology

Rename core concepts for better approachability:

- "mission" → "workflow template"
- "pack" → "team preset"
- "free routing" → "custom mode"
- "packet" → "task description"
- "verdict" → "review result"
- "escalation" → "help request"
- "role contract" → "role profile"

Rationale: New users find the current terminology confusing. Terms like "mission," "verdict," and "escalation" sound military or legal, which can be off-putting. More approachable alternatives will lower the barrier to adoption.
