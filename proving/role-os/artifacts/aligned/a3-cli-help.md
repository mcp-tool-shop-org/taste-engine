roleos run <task>        Create a persistent run. Auto-decides mission vs pack vs free routing.
roleos next             Start the next step in the active run.
roleos complete <file>  Complete the active step with an artifact file.
roleos explain          Show full run state, current step guidance, and pending work.
roleos escalate <from> <to> <reason> <action>  Route blocked work to the right resolver.
roleos friction         Measure operator touches and intervention frequency.
